---
description: how to add admin users like Accountant, HR, etc. (System Access Workflow)
---

To grant administrative access to the StoreAI Intelligence system:

1. **Access Identity Hub**:
    - Go to **"System Core"** in the sidebar (accessible only by Super Admins).
    - Note: If "System Core" is not visible, ensure you are logged in as the Super Admin.

2. **Generate New Identity**:
    - Click **"New User"** (or Plus icon) in the top header.

3. **Configure Permissions**:
    - **Email/Password**: Set their secure login credentials.
    - **Access Role**: Choose the specific operational permission level:
        - **ACCOUNTANT**: Access to Financial Ledger and Accounts.
        - **HR MANAGER**: Access to Workforce Portal and Attendance.
        - **SUPERVISOR**: Access to Inventory and Staff.
        - **POS_ADMIN**: Restricted access to Sales Terminal only.
        - **ADMIN**: Global access.

4. **Activation**:
    - Click **"Commit Protocol"**. The user can now log in immediately.
    - To link this user to a payroll record, follow the **Staffing Workflow**.
